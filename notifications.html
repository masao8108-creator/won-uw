카카오톡 알림톡 연동
상담 리마인더, 계약 안내, 청약 알림을 고객에게 카카오톡으로 자동 발송합니다.
⚡ 설정 필요
⚠️ 카카오 알림톡 발송 구조:
카카오 비즈니스 채널 등록 → KakaoTalk Channel (카카오톡 채널) 생성 → 알림톡 API (카카오 i 오픈빌더 또는 솔루션사) 연동
빠른 테스트를 원하시면 아래 카카오 나에게 보내기 API (개인 테스트용)를 먼저 사용하세요.
나에게 보내기 (테스트용)
카카오 개발자 계정의 REST API 키로 본인에게만 발송 테스트가 가능합니다. 실제 고객 발송은 아래 알림톡 API를 사용하세요.
알림톡 API (실 서비스)
솔루션사(알리고, 쿨SMS, 문자나라 등)의 알림톡 API를 사용하면 고객 번호로 알림톡을 발송할 수 있습니다.
