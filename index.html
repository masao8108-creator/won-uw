GitHub Pages 배포 가이드
🚀 무료 공개 URL로 배포하기
GitHub Pages를 이용하면 설치 없이 어느 기기에서나 접속 가능한 URL이 생깁니다.
https://[계정명].github.io/won-uw/ 형태로 배포됩니다.
1
GitHub 계정 생성 및 저장소 만들기
github.com 접속 → 회원가입 → New repository → 이름: won-uw → Public 선택 → Create
2
4개 파일 업로드
저장소 페이지에서 "uploading an existing file" 클릭 → 아래 4개 파일을 드래그앤드롭
index.html   goji_analyzer.html   underwriting.html   customer_db.html 복사
3
GitHub Pages 활성화
저장소 → Settings → Pages → Source: "Deploy from a branch" → Branch: main → / (root) → Save
4
배포 URL 확인 (약 2~3분 후)
Pages 설정 화면에 URL이 표시됩니다. 이 URL을 설계사들과 공유하면 바로 사용 가능합니다.
https://[계정명].github.io/won-uw/ 복사

주의 — API 키 설정
