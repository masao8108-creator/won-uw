✅ 아래 순서대로 5~10분이면 설정 완료됩니다. 한번만 하면 이후로는 드라이브에 파일만 올리면 자동 반영됩니다.
STEP 1 — Google Cloud 프로젝트 생성
1. console.cloud.google.com 접속 → Google 계정 로그인
2. 상단 프로젝트 선택 → 새 프로젝트 → 이름: won-uw → 만들기
STEP 2 — Google Drive API 활성화
1. 왼쪽 메뉴 → API 및 서비스 → 라이브러리
2. 검색창에 Google Drive API 입력 → 클릭 → 사용 설정
STEP 3 — OAuth 동의 화면 설정
1. API 및 서비스 → OAuth 동의 화면
2. 사용자 유형: 외부 → 만들기
3. 앱 이름: 원금융 UW / 이메일 입력 → 저장 후 계속
4. 범위 추가 → ../auth/drive.readonly 검색 → 추가
5. 테스트 사용자에 본인 이메일 추가 (중요!)
STEP 4 — OAuth 클라이언트 ID 발급
1. 사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
2. 애플리케이션 유형: 웹 애플리케이션
3. 승인된 JavaScript 원본에 아래 주소 추가:
https://[계정명].github.io 복사
4. 만들기 → 클라이언트 ID 복사 → 설정 탭에 입력
STEP 5 — API 키 발급
1. 사용자 인증 정보 만들기 → API 키
2. 생성된 키 복사 → 설정 탭에 입력
3. (권장) API 키 제한 → HTTP 리퍼러: *.github.io/*
STEP 6 — 드라이브 폴더 준비 & ID 복사
1. Google Drive에서 폴더 생성: 원금융_인수심사자료
2. 폴더 URL 확인: drive.google.com/drive/folders/여기가 폴더 ID
3. 폴더 ID 복사 → 설정 탭에 입력
